HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. WEb browsers recieve HTML documents from a web sever or form local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally inculded cues for the appearance of the document.
